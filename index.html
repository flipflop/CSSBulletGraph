Bullet Graph CSS

A simple, pure CSS Bullet Graph by @terraflop

Motivation

On a number of past projects I have been asked to create a project or development dashboard to quickly visualise measurements and comparisons. Typically complex information of this type is conveyed via guages, meters and donut charts. After researching a number of Data Visualisation alternatives, Bullet Graphs seemed to suit my requirements perfectly.

After first deciding the write a small JavaScript library to create Bullet Graphs, I quickly discovered that CSS could handle the display entirely.

Features

  • CSS API
  • Pure CSS
  • Sans JS

CSS API

  • .bar.width
  • .bar.color
  • .marker
  • .marker.distance
  • .bulletgraph .background .step.color
  • .bulletgraph .background .step.width

Planned Features
